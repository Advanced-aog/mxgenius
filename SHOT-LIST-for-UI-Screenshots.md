### 4. Parts Search Screenshot

### 5. Support Interface Screenshot

## 📐 TECHNICAL specs

### Screenshots
- Resolution: 1920x1080 minimum
- Format: PNG or JPG
- File naming: `screenshot-[section]-[date].png`

### Screen Recordings
- Resolution: 1920x1080
- Frame rate: 30fps
- Format: MP4
- Include cursor in recordings
- Audio: Include system audio if showing AI responses

### B-Roll
- Resolution: 4K preferred, 1080p minimum
- Format: MP4
- Frame rate: 24-30fps
- Good lighting important
